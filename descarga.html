Descarga

DaVinci Concurrente es una aplicación escrita en el lenguaje de prorgamación Java, por lo que requiere que tengas instalado el JRE 1.7.X+

Si no sabes si tenes Java instalado, podes comprobarlo utilizando el siguiente enlace www.java.com/verify

Si no tenes Java instalado, podes descargarlo utilizando el siguiente enlace www.java.com/getjava

Si tenes Java instalado, utilizás windows y haciendo click en ejecutables de java (.jar) no se ejecutan, desgarga la aplicación jarfix del siguiente enlace http://johann.loefflmann.net/en/software/jarfix/index.html
